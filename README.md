# NodeJS TypeScript Starter

## Features
* NodeJS 6.x
* Express 4.x
* TypeScript 1.8
* Debugging Integration for Visual Studio Code
* Unit-Testing with Jasmine

* Linting?

## Installation

1. Install NodeJS > v5.0.0
2. `npm install -g nodemon typescript typings jasmine watch`
3. `typings install`
